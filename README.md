# 🧠 BitNet: Transformer Variant from Scratch (Custom Implementation)

This repository contains a **from-scratch implementation of BitNet**, a transformer variant known for its efficient architecture using binary-weighted matrices, significantly reducing computational overhead while maintaining performance. 
### NOTE: This repo is still in working phase.

## 📓 Notebook: `bitnet_custom.ipynb`

This Jupyter notebook provides:
- A clean, custom implementation of BitNet without relying on HuggingFace or high-level libraries.
- An educational walkthrough of BitNet’s internal components.
- End-to-end forward pass demonstration with synthetic data.

---

## 🚀 Features

- 🔧 **Manual Implementation**: All core modules (BitLinear, BitNetBlock, etc.) are coded from scratch.
- 🧮 **Binary Matrix Operations**: Leverages binary approximations for linear transformations.
- 🧠 **Custom Forward Pass**: Showcases how BitNet performs on toy data input.
- 📈 **Parameter Count Logging**: Displays trainable parameters of the network.
- 💡 **Educational Structure**: Great for learning advanced transformer variants.

---

## 📂 Requirements

- Python ≥ 3.8

- PyTorch ≥ 1.12

- Jupyter Notebook

## Install Dependencies: pip install torch numpy jupyter

## Usage:

1. Clone the repository: git clone github_repo_path
2. cd bitnet-from-scratch
3. jupyter notebook bitnet_custom.ipynb / Run the cells individually for better understanding
